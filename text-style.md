# 📌 テキスト装飾 (Text Style)

## 書き方

- 「**太字**」
```markdown
**太字** or 「Ctrl + B」
```

- 「_Italic_(斜体)」
```markdown
_Italic_ or 「Ctrl + I」
```
🚨 **※現状、日本語の斜体表示はできないようなので注意**

https://github.com/missing-semester-jp/missing-semester-jp.github.io/issues/18

- 「**_BOLD + Italic_**(斜体 + 太字)」
```markdown
**_BOLD + Italic_**
```

- 「~~打ち消し線~~」
```markdown
~~打ち消し線~~
```
- 「`インラインコード`」
```markdown
`インラインコード`
```

- 「<sup>上付き文字</sup>」
```markdown
<sup>上付き文字</sup>
```
- 「<sub>下付き文字</sub>」
```markdown
<sub>下付き文字</sub>
```
- 「<ins>下線</ins>」
```markdown
<ins>下線</ins>
```
